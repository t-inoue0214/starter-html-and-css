# 02. HTML の主要なタグ

ここでは、Web ページを作る上で頻繁に使用する基本的なタグを学習します。

## 1. ブロックレベル要素とインライン要素

HTML のタグには、大きく分けて「箱（ブロック）」として扱われるものと、「行内（インライン）」として扱われるものがあります。

※ HTML5 以降、厳密な定義は変わりましたが、初心者のうちはこのイメージを持つことが重要です。

- **ブロックレベル（積み木のように縦に積まれる）**

  - 見出し (`h1`~`h6`)、段落 (`p`)、リスト (`ul`, `li`)、区画 (`div`) など

- **インライン（文字のように横に並ぶ）**

  - リンク (`a`)、画像 (`img`)、強調 (`strong`)、スパン (`span`) など

## 2. 重要概念：タグの役割 — セマンティックタグ vs. レイアウト用タグ

HTML を書く時、「なぜそのタグを使うのか」が重要です。

大きく分けて 2 種類のタグがあります。

1. **意味を持つタグ（セマンティック）**

    ブラウザや検索エンジンに「これは見出しだ」「これは箇条書きだ」と正体を伝えるタグです。

    Web ブラウザのエンジンごとにデフォルトのスタイルを持っているため、何もしなくてもフォントサイズや周辺のスペースが変わることがあります。

    - `<h1>`: ここタイトルです！
    - `<p>`: ここは文章です！
    - `<button>`: ここは押せます！

1. **レイアウト・装飾のためのタグ（汎用的な箱）**

    ただの「入れ物」です。

    それ自体に意味はありませんが、「レイアウトのため」や「CSS で色をつけるため」にグループ化したい時に使います。

    | タグ     | 分類       | 説明                                                                                                         |
    | -------- | ---------- | ------------------------------------------------------------------------------------------------------------ |
    | `<div>`  | ブロック   | Division（区分）。要素をまとめてグループ化するための「大きな箱」です。レイアウトを作る際によく使います。 |
    | `<span>` | インライン | Span（範囲）。文章の一部だけを赤くしたい時などに、その範囲を囲むための「透明な袋」です。                 |

> **使い分けのコツ**  
> 基本は「意味のあるタグ」を使います。それらで表現できないレイアウトや装飾が必要になった時だけ、`<div>` や `<span>` を使います。

## 3. よく使うタグ一覧

各タグの詳しい仕様は、**MDN Web Docs** へのリンクを貼っています。

演習を始める前に、使うタグの MDN ページを一度開いてみましょう。ページ上部の構文例と「例」セクションを見るだけでも OK です。

### 見出しと文章

| タグ            | 説明                                                                              | MDN リンク                                                                      |
| :-------------- | :-------------------------------------------------------------------------------- | :------------------------------------------------------------------------------ |
| `<h1>` ~ `<h6>` | **見出し (Heading)** <br>h1 が一番大きく、h6 が小さい。h1 はページに 1 つが基本。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/Heading_Elements) |
| `<p>`           | **段落 (Paragraph)** <br>文章のまとまりを表します。                               | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/p)                |
| `<br>`          | **改行 (Break)** <br>強制的に改行します。閉じタグは不要です。<br>詩や住所など、改行位置に意味がある場合のみ使います。段落の区切りには `<p>` を使いましょう。`<br>` を 2 つ連続して使うのは誤りです。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/br)               |
| `<strong>`      | **重要 (Strong)** <br>重要な語句を示します。太字で表示されます。見た目のためだけではなく「意味として重要」な箇所に使います。スクリーンリーダーは `<strong>` を「強い重要性」として読み上げる場合があります。単に太字にしたいだけなら CSS を使うべきで、`<b>` タグと混同しないように注意。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/strong)           |
| `<em>`          | **強調 (Emphasis)** <br>強調を示します。斜体で表示されます。`<i>` タグと混同しないように注意。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/em)               |

### リスト（箇条書き）

| タグ   | 説明                                                                  | MDN リンク                                                        |
| :----- | :-------------------------------------------------------------------- | :---------------------------------------------------------------- |
| `<ul>` | **順序なしリスト (Unordered List)** <br>・(中黒)で表示されるリスト。  | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/ul) |
| `<ol>` | **順序付きリスト (Ordered List)** <br>1, 2, 3... と番号がつくリスト。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/ol) |
| `<li>` | **リスト項目 (List Item)** <br>ul や ol の中で使います。              | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/li) |

### リンクと画像

| タグ    | 説明                                                                                                | MDN リンク                                                         |
| :------ | :-------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------- |
| `<a>`   | **アンカー (Anchor)** <br>他のページへのリンクを作成します。`href` 属性で URL を指定します。<br>外部サイトを新しいタブで開く場合は `target="_blank"` と `rel="noopener noreferrer"` をセットで書くのが実務のルールです（セキュリティ上の理由）。詳細は 03章で学びます。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/a)   |
| `<img>` | **画像 (Image)** <br>画像を表示します。`src` 属性でファイルの場所を指定します。`<img>` は内容を持たない **void element（空要素）** のため、閉じタグは不要です。ただし XML との互換性を意識して `<img ... />` と `/>`で閉じる書き方も多く見られます。本教材ではこの書き方に統一します。<br>`alt` 属性に画像の内容を表すテキストを必ず書きます（アクセシビリティと SEO のために重要）。装飾目的の画像は `alt=""` と空にします。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/img) |

### ページ構造タグ

| タグ       | 説明                                                                                    | MDN リンク                                                            |
| :--------- | :-------------------------------------------------------------------------------------- | :-------------------------------------------------------------------- |
| `<header>` | **ヘッダー (Header)** <br>ページや記事の導入部分を表します。ロゴ・タイトルなどを入れます。   | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/header) |
| `<nav>`    | **ナビゲーション (Navigation)** <br>サイト内リンクやメニューなど、主要なナビゲーションリンクをまとめます。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/nav)    |
| `<main>`   | **メインコンテンツ (Main)** <br>ページのメインとなるコンテンツ領域を表します。1 ページに 1 つ。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/main)   |
| `<footer>` | **フッター (Footer)** <br>ページの末尾情報（著作権・連絡先など）を表します。               | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/footer) |

## 演習課題：自己紹介ページを完成させよう

`index.html` を開き、以下の手順に従ってコードを修正・追記してください。

（Web サーバーを起動し、ブラウザで確認しながら進めましょう）

### 手順 1: 見出しの修正

現在、`<h1>` タグの中身が「名前（ここに名前を入れる）」になっています。

ここを **あなたの名前** に書き換えてください。

ブラウザで確認し、ページ上部に自分の名前が大きく表示されれば正解です。

### 手順 2: プロフィールの追記

「私のプロフィール」という `<h2>` タグの下に、`<p>` タグを使って簡単な自己紹介文を書いてください。

`<p>` タグを複数書いて、自己紹介を複数の段落に分けてみましょう。

ブラウザで確認し、自己紹介文が段落ごとに分かれて表示されれば正解です。

### 手順 3: 好きな食べ物リストの作成

「好きな食べ物」という `<h2>` タグの下に、`<ul>` と `<li>` を使って、好きな食べ物を 3 つ箇条書きでリストアップしてください。

ブラウザで確認し、3 つの項目が「・」付きのリストで表示されれば正解です。

### 手順 4: リンクの作成

`<footer>` の中にはリンクのひな型が用意されています。確認してみましょう。

その上でひな型を参考に、`<a>` タグの `href` 属性を書き換えたり、リンクテキストを変えたりして、よく見る Web サイト（Google や Yahoo など）へのリンクを自分で作成してみましょう。

ブラウザで確認し、フッター内にリンクが表示され、クリックするとページへ移動できれば正解です。

### 手順 5: 画像の alt テキストを修正しよう

index.html の `<img>` タグを確認してください。

現在の `alt="ダミー画像"` を、実際に何の画像かを説明するテキストに書き換えてみましょう。

## 覚えなくて良いタグ・後回しにするタグ

HTML のタグはたくさんありますが、初心者がすべて覚える必要はありません。

1. **もう使ってはいけないタグ（見た目タグ）**

    昔の HTML では使われていましたが、現在は 「見た目は CSS でやるべき」 というルールになったため、非推奨（または廃止）になったタグです。

    - `<font>`: 文字色やサイズを変える -> CSS でやるので不要
    - `<center>`: 中央寄せする -> CSS でやるので不要
    - `<big>`: 文字サイズ変更 -> CSS でやるので不要

1. **今は覚えなくて良いタグ（高度・特殊）**

    必要になった時に調べれば OK です。

    - `<small>`: 著作権表示などに使います。最初は覚えなくて OK です。
    - `<table>`: 表組みを作ります。レイアウト目的で使ってはいけません。
    - `<canvas>`, `<svg>`: 図形描画に使います。
    - `<audio>`, `<video>`: 音声や動画再生に使います。
    - `<iframe>`: 別のサイト（Google マップなど）の埋め込みで使ったりします。安易に使うのはご法度
