# 02. HTML の主要なタグ

ここでは、Web ページを作る上で頻繁に使用する基本的なタグを学習します。

---

## 1. ブロックレベル要素とインライン要素

HTML のタグには、大きく分けて「箱（ブロック）」として扱われるものと、「行内（インライン）」として扱われるものがあります。

※ HTML5 以降、厳密な定義は変わりましたが、初心者のうちはこのイメージを持つことが重要です。

- **ブロックレベル（積み木のように縦に積まれる）**
  - 見出し (`h1`~`h6`)、段落 (`p`)、リスト (`ul`, `li`)、区画 (`div`) など
- **インライン（文字のように横に並ぶ）**
  - リンク (`a`)、画像 (`img`)、強調 (`strong`)、スパン (`span`) など

## 2. 重要概念：タグの「意味」と「無意味」

HTML を書く時、「なぜそのタグを使うのか」が重要です。大きく分けて 2 種類のタグがあります。

① **意味を持つタグ（セマンティック）**<br>
ブラウザや検索エンジンに「これは見出しだ」「これは箇条書きだ」と正体を伝えるタグです。<br>
Web ブラウザのエンジンごとにデフォルトのスタイルを持っているため、何もしなくてもフォントサイズや周辺のスペースが変わることがあります。

- `<h1>`: ここは重要です！
- `<p>`: ここは文章です！
- `<button>`: ここは押せます！

② **意味を持たないタグ（汎用的な箱）**<br>
ただの「入れ物」です。<br>
それ自体に意味はありませんが、「レイアウトのため」や「CSS で色をつけるため」にグループ化したい時に使います。

| タグ     | 分類       | 説明                                                                                                         |
| -------- | ---------- | ------------------------------------------------------------------------------------------------------------ |
| `<div>`  | ブロック   | Division（区分）。<br>要素をまとめてグループ化するための「大きな箱」です。レイアウトを作る際によく使います。 |
| `<span>` | インライン | Span（範囲）。<br>文章の一部だけを赤くしたい時などに、その範囲を囲むための「透明な袋」です。                 |

> **使い分けのコツ**<br>
> 基本は「意味のあるタグ」を使います。それらで表現できないレイアウトや装飾が必要になった時だけ、`<div>` や `<span>` を使います。

## 3. よく使うタグ一覧

各タグの詳しい仕様は、 **MDN Web Docs** へのリンクを貼っています。リンク先を参照してください。

### 見出しと文章

| タグ            | 説明                                                                              | MDN リンク                                                                      |
| :-------------- | :-------------------------------------------------------------------------------- | :------------------------------------------------------------------------------ |
| `<h1>` ~ `<h6>` | **見出し (Heading)** <br>h1 が一番大きく、h6 が小さい。h1 はページに 1 つが基本。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/Heading_Elements) |
| `<p>`           | **段落 (Paragraph)** <br>文章のまとまりを表します。                               | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/p)                |
| `<br>`          | **改行 (Break)** <br>強制的に改行します。閉じタグは不要です。                     | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/br)               |

### リスト（箇条書き）

| タグ   | 説明                                                                  | MDN リンク                                                        |
| :----- | :-------------------------------------------------------------------- | :---------------------------------------------------------------- |
| `<ul>` | **順序なしリスト (Unordered List)** <br>・(中黒)で表示されるリスト。  | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/ul) |
| `<ol>` | **順序付きリスト (Ordered List)** <br>1, 2, 3... と番号がつくリスト。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/ol) |
| `<li>` | **リスト項目 (List Item)** <br>ul や ol の中で使います。              | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/li) |

### リンクと画像

| タグ    | 説明                                                                                                | MDN リンク                                                         |
| :------ | :-------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------- |
| `<a>`   | **アンカー (Anchor)** <br>他のページへのリンクを作成します。`href` 属性で URL を指定します。        | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/a)   |
| `<img>` | **画像 (Image)** <br>画像を表示します。`src` 属性でファイルの場所を指定します。閉じタグは不要です。 | [詳細](https://developer.mozilla.org/ja/docs/Web/HTML/Element/img) |

## 演習課題：自己紹介ページを完成させよう

`index.html` を開き、以下の手順に従ってコードを修正・追記してください。<br>
（Web サーバーを起動し、ブラウザで確認しながら進めましょう）

### 手順 1: 見出しの修正

現在、`<h1>` タグの中身が「名前（ここに名前を入れる）」になっています。<br>
ここを **あなたの名前** に書き換えてください。

### 手順 2: プロフィールの追記

「私のプロフィール」という `<h2>` タグの下に、`<p>` タグを使って簡単な自己紹介文を書いてください。<br>
途中で `<br>` を使って改行を入れてみましょう。

### 手順 3: 好きな食べ物リストの作成

「好きな食べ物」という `<h2>` タグの下に、`<ul>` と `<li>` を使って、好きな食べ物を 3 つ箇条書きでリストアップしてください。

### 手順 4: リンクの作成

一番下の `<footer>` の中に、`<a>` タグを使って、よく見る Web サイト（Google や Yahoo など）へのリンクを作成してください。

## 覚えなくて良いタグ・後回しにするタグ

HTML のタグはたくさんありますが、初心者がすべて覚える必要はありません。

1. **もう使ってはいけないタグ（見た目タグ）**<br>
   昔の HTML では使われていましたが、現在は 「見た目は CSS でやるべき」 というルールになったため、非推奨（または廃止）になったタグです。

- `<font>`: 文字色やサイズを変える -> CSS でやるので不要
- `<center>`: 中央寄せする -> CSS でやるので不要
- `<big>`, `<small>`: 文字サイズ変更 -> CSS でやるので不要

2. **今は覚えなくて良いタグ（高度・特殊）**<br>
   必要になった時に調べれば OK です。

- `<table>`: 表組みを作ります。レイアウト目的で使ってはいけません。
- `<canvas>`, `<svg>`: 図形描画に使います。
- `<audio>`, `<video>`: 音声や動画再生に使います。
- `<iframe>`: 別のサイト（Google マップなど）の埋め込みで使ったりします。安易に使うのはご法度
